## 1.0.0 - 2026-04-14

* Initial release of the heypster Flutter SDK.
* `HeypsterFlutterSDK.configure()` — SDK initialization with API key.
* `HeypsterDialog` — Pre-built GIF picker with search, emotion browsing,
  and trending GIFs.
* `HeypsterMediaView` — Single GIF display widget with MP4 video playback.
* `HeypsterGridView` — Scrollable paginated grid driven by content requests.
* GIPHY-compatible API support (search, trending, random, translate, emoji).
* Native heypster API support (emotions, tags, badges, GIFs of the day).
* 27 emotion categories for browsing.
* 11 language support for tag search.
* In-memory LRU response cache with configurable cache policy.
